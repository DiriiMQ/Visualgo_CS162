This program will visualize some basic data structures, including animations (similar to \href{https://visualgo.net/en/list}{\texttt{ Visualgo.\+net}}). The data structures are\+:
\begin{DoxyItemize}
\item Linked List
\begin{DoxyItemize}
\item Singly Linked List
\item Doubly Linked List
\item Circular Linked List
\end{DoxyItemize}
\item Stack
\item Queue
\item \mbox{\hyperlink{classArray}{Array}}
\begin{DoxyItemize}
\item Static \mbox{\hyperlink{classArray}{Array}}
\item Dynamic \mbox{\hyperlink{classArray}{Array}}
\end{DoxyItemize}
\end{DoxyItemize}

The program is written in C++ and uses SFML library for graphics. The features of this project are followed from \href{./docs/CS162-CSC10002-Solo\%20Project.pdf}{\texttt{ this requirements}}.\hypertarget{md_README_autotoc_md1}{}\doxysection{How to run}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item C++ 17 – GNU GCC 7.\+3.\+0+
\begin{DoxyItemize}
\item For Windows users, please follow \href{https://www.msys2.org/\#installation}{\texttt{ this instruction}} to install.
\begin{DoxyItemize}
\item {\bfseries{Note}}\+: Please install the {\ttfamily Min\+GW 64-\/bit} version. At step 6, please run the following command instead\+:
\begin{DoxyItemize}
\item {\ttfamily pacman -\/S mingw-\/w64-\/x86\+\_\+64-\/gcc}
\end{DoxyItemize}
\end{DoxyItemize}
\item For Debian users, run the following command\+:
\begin{DoxyItemize}
\item {\ttfamily sudo apt-\/get update \&\& sudo apt-\/get install build-\/essential}
\end{DoxyItemize}
\item For Mac users, run the following command\+:
\begin{DoxyItemize}
\item {\ttfamily xcode-\/select -\/-\/install}
\end{DoxyItemize}
\end{DoxyItemize}
\item \href{https://cmake.org/download/}{\texttt{ CMake 3.\+26+}}
\item Makefile
\begin{DoxyItemize}
\item For Windows users, please open the {\ttfamily MSYS2 Min\+GW 64-\/bit} terminal and run the following command\+:
\begin{DoxyItemize}
\item {\ttfamily pacman -\/S make}
\end{DoxyItemize}
\item For Debian users, run the following command\+:
\begin{DoxyItemize}
\item {\ttfamily sudo apt-\/get install make}
\end{DoxyItemize}
\item For Mac users, {\ttfamily make} is already installed when you run the above command.
\end{DoxyItemize}
\item SFML 2.\+5+
\begin{DoxyItemize}
\item For Windows users, please open the {\ttfamily MSYS2 Min\+GW 64-\/bit} terminal and run the following command\+:

\item For Debian users, run the following command\+:
\begin{DoxyItemize}
\item {\ttfamily sudo apt-\/get install libsfml-\/dev}
\end{DoxyItemize}
\item For Mac users, please follow \href{https://www.sfml-dev.org/tutorials/2.5/start-osx.php\#installing-sfml}{\texttt{ this instructions}} to install.

\begin{DoxyItemize}
\item The program will show the main screen when it starts.
\end{DoxyItemize}




\begin{DoxyItemize}
\item Select a data structure by clicking on the buttons on the top-\/left of the screen.
\item Do the operations on the data structure by clicking on the arrow button on the left side of the screen.
\item Read the demo code by clicking on the arrow button on the right side of the screen.
\end{DoxyItemize}




\begin{DoxyItemize}
\item Example for using the creating mode\+:
\end{DoxyItemize}




\begin{DoxyItemize}
\item If you choose the other modes, it will show the animation of the operations on the data structure and demonstrate the demo code at the same time.
\end{DoxyItemize}




\begin{DoxyItemize}
\item Run the animation by clicking on the play/pause button in the controller.
\item Next step by clicking on the next button in the controller.
\item Previous step by clicking on the previous button in the controller.
\end{DoxyItemize}




\begin{DoxyItemize}
\item You can change the speed of the animation by clicking on the speed button in the controller.
\end{DoxyItemize}
